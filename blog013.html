    perl练习，源码

    #!/usr/bin/perl -w

$zhu = 1;

do{
    my %red;                         #hash
    for (0..5) {
        my $no;
        do {
            $no = int rand(33) +1;  # int [1:33]         
        } while (exists $red{$no}); # $no is hash's key
        	   $red{$no} = 1;
    }

    my $blue=int rand(16) +1;       #int [1:16]
          
    print join("\t", keys %red, "$blue"), "\n\n";

    $zhu = $zhu +1;   #initial 5 zhu

}while($zhu <= 5);

    知识点

    1. rand(16)表示生成0-15的随机数，要生成1-16直接增加偏移量+1即可。
    2. hash的用法：这里用到了hash的exists函数检测key，让随机产生的数不同。
      这里是指，如果hash%red中存在key$no，则返回再生成一次随机数；
