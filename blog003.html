    3--perl生成html table报告-项目3.3

    项目 > 项目3-Perl回归测试脚本 perl
    发布时间 : 2023-03-17 23:22
    字数:651 评论:
    1. 一、主要功能
    2. 二、html头文件代码
    3. 三、$table中的html table代码
    4. 四、html table CSS美化代码
    5. 五、总结

    一、主要功能

    • 生成html table
    • CSS美化表格

    二、html头文件代码

    在perl中定义$html变量，当生成好html代码后，通过文件句柄导出$html中所有内容到verification.html文件，就可以firefox verification.html &看到最终结果的表格文件。

    my $html = "
<html lang='en'>

    <head>
        <meta charset='utf8' />
        <link rel='stylesheet' href='css/styles.css'>
        <title>
            verification table
        </title>
    </head>

    <style>
    $CSS       ##### CSS 美化代码
    </style>

    <body>
  <table border=1>
      <tr>
       <th>Test Case </th>
       <th>Description </th>
       <th>Dir </th>
       <th>Result </th>
       <th>Time </th>
      </tr>
      $table       ### 表格一行内容的 html table代码
  </table>  
    </body>

</html>
";

##########---------------export html and clear some log
open(FILE,">verification.html");
say FILE "$html";
close (FILE);

    三、$table中的html table代码

    将for循环产生的相关单元格变量填入html table中相应位置，每一次for循环都会产生一行数据

    $table .= "<tr>\n\t\t
              <td><a href= \"../tb/$case_name.v\">test_$i</a></td>\n\t\t
              <td>$description</td>\n\t\t
              <td><a href= \"**$dir_log\**">log_file_test_$i</a></td>\n\t\t
              <td bgcolor= $color > $result</td>\n\t\t
              <td>$time</td>\n
           </tr>\n"

    四、html table CSS美化代码

    #########-------------------------CSS for table style
$CSS='
table {
    width: 90%;
    background: #ccc;
    margin: 10px auto;
    border-collapse: collapse;
    border: 2px inset black;
}
th,
td {
    height: 25px;
    line-height: 25px;
    text-align: center;
    border: 1px solid #ccc;
}
th {
    background: #eee;
    font-weight: bold;
    font-size:15px;
}
tr {
    background: #fff;
}
tr:hover {
    background: #cc0;
}
td a {
    color: #06f;
    text-decoration: none;
}
td a:hover {
    color: #06f;
    text-decoration: underline;
}
';

    注意，这里使用的单引号对$CSS变量赋值，防止其中内容在双引号中当作perl的关键字被识别导致语法错误。

    五、总结

    到这，此次简单的perl回归测试报告脚本就完成了。这算是学完perl后，第一次写的一个原创的perl脚本，中间的逻辑可能不是最优解，但好歹能完成基本功能。通过这个脚本的书写，我第一次体会到了perl的强大和复杂，也深知自己之前仅仅学习了皮毛而已。同时我也了解了html、CSS相关的一些网页知识，这和我搭建blog的想法正不谋而合，高级的用法虽然不会，但底层逻辑大概了解了，算是打开了又一个世界的大门了！
